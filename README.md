## Overview

The *Dynamic Code Generation* library is set of classes that assists with creating and implementing types at runtime. This is the base to build powerful features that require code referring to types that are not known at compilation time. 

## Supported Platforms

The *WPF Toolkit* is entirely written in C# and should work with .NET Framework 4.6.1 or higher.
